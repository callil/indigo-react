## `<Icon />`
Displays an icon.

#### Props
```ts
PositionProps &
Layout &
SpaceProps & {
  icon: string;
  fill: string;
};
```

#### Default Props
```js
{
  icon: 'NullIcon',
  fill: '#000000'
}
```

#### Usage
```tsx
<Icon icon='ChevronWest' />
```
